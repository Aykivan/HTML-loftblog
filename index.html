Разметка страницы

Теги разметки текста будут представлены ниже.

    Среди них:
    • Тег абзатца p
    • Теги жирности b и strong
    • Теги курсива i и em
    • Теги нижнего регистра sub и верхнего sup
    • Тег выделения желтым mark

HTML (HyperText Markup Language — «язык гипертекстовой разметки») — самый базовый строительный блок Веба. Он определяет содержание и структуру веб-контента. Другие технологии, помимо HTML, обычно используются для описания внешнего вида/представления (CSS) или функциональности/поведения (JavaScript) веб-страницы.

Под гипертекстом("hypertext") понимаются ссылки, которые соединяют веб-страницы друг с другом либо в пределах одного веб-сайта, либо между веб-сайтами. Ссылки являются фундаментальным аспектом Веба. Загружая контент в Интернет и связывая его со страницами, созданными другими людьми, вы становитесь активным участником Всемирной паутины.

Выше представлены варианты выделения текста жирным и курсивом. При помощи этих способ текс выделен лишь визуально, если мы хотим расставить акцент на нем стоит использовать следующие способы: ЖИРНЫЙ и КУРСИВ.

Списки

Не нумерованные списки используют теги ul:

  • Первый
  • Второй
  • Третий

Нумерованные списки используют теги ol:

  1. Первый
  2. Второй
  3. Третий

В нумерованных списках есть 3 атрибута:

  1. reversed - обратная нумерация
  2. start ='' нумерация с определенной цифры
  3. type ='' задает вид нумерации, например I

Ссылки

Для создания ссылки используется тег а вместе с атрибутом, который указывает, куда ведет ссылка - href.

В данном случае использовался абсолютный адресс. Он имеет:

  • Протокол - https
  • Имя сервера - www.youtube.com
  • Его адресс - все что за /
Так же существует и относительный адресс, он открывает файлы относительно стартовой точки, в нем нет протокола и адресса сайта.

Локальные ссылки с использованием относительных адресов:

  • Ссылка на файл в той же папке
  • Ссылка на файл в соседней папке
  • Ссылка на файл в папке уровнем выше представлена в ссылке на соседней папке

Для того, что бы cсылка открывалась не на нашем сайте, а в новой вкладке, необходимо добавить атрибут target="_blank"

Существуют три вида ссылок:

  1. Внешние - те которые мы создали выше
  2. Якоря - те, которые ведут нас в какую-то часть документа, они создаются через атрибут - id с уникальным именем, на этом принципе основанна следующая ссылка, которая приведет нас к разделу: "Разметка страницы". Так же можно ссылаться на якоря на других страницах, для этого необходимо сперва указать ссылку на сайт, а затем #имяякоря
  3. Ссылка на скачивание. Если поле атрибута download оставить пустым, файл будет скачан с именем, которое есть у файла, если мы хотим изменить это, нужно прировнять к атрибуту то имя, которое нам нужно.

Медиа

Атрибуты src и alt являются обязательными, при использовании изображений. С помощью атрибута width можно задать размер картинки, если мы не указываем единицы измерения, они по умолчанию будут в px.

С помощью тега video, мы можем добавить видео файл, он является парным тегом. Это сделано для того чтобы дать возможность загружать видео разных форматов с помощью атрибута source, которые поддерживают разные браузеры. С помощью атрибута type мы задаем формат видео. И если мы хотим дать возможность управлять показом видео, необходимо добавить атрибут - controls. Аналогично и аудиофайлами, только вместо тега video используется тег - audio.

Для ссылки на видео с youtube, необходимо скопировать специальную ссылку на самом хостинге. Атрибут frameborder="10" задает обрамление для плеера.

С помощью тега iframe можно так же добавлять и карты.

Формы №1





Забыли пароль?

В формах есть ряд тегов:

  • action - тег с адресом программы, которая будет обрабатывать форму после отправки
  • method - get или post - сообщает серверу о методе запроса
    • get - передача через url (адресные строки)
    • post - метод скрытой отправки данных.
  • imput - тег поля ввода, одиночный
    • type - тип вводимых данных. test, password (закрывает вводимый текст звездочками)
    • name - имя для понимания браузером, что мы ввели
  • label - парный тег, выводит на экран необходимый текст. Есть два варианта оформление, либо вписать в него imput, либо через атрибут for, которому присваивается значение уникального id.
  • br - тег для создания пустой строки, временное решение пока мы не знаем разметки и CSS
  • submit - атрибут от input, для отправки формы, чтобы изменить его имя, добавляем его атрибут - value
  • value - может работать во всех типах imput, вводя в те окна значения по умолчанию
  • button - парный тег отправки, отличий со способом отправки через imput нет, преиммущество парного тега - возможность сложной стилизации
Отправить форму так же можно и без дополнительных тегов, используя - enter.

Новые теги:

  • placeholder - подсказка в виде примера заполнения графы
  • type = number - принимает в себя только числа
  • type = email - принимает на вход только корректные адресса почты
  • type = radio - радиокнопки, в виде окошка, которое модно раскрыть или свернуть, указав для них одинаковые именамы можем выбрать только одно значение
  • checked - атрибут, устанавливающий значение по умолчанию
  • fieldset - тег для обьединения нескольких label
  • legend - тег для создания общего заголовка в блоке - fieldset
  • select - тег для создания окна выбора
  • option - тег для записи вариантов внутри select
  • multiple - атрибут, позволяющий сделать несколько вариантов выбора в select
  • input type="checkbox" - создает меню с возможностью поставить галочки в ячейки
  • textarea - парный тег, необходимый для многострочных текстов
  • type = "file" - позволяет нам загружать файлы на сайт
